# NexusAds Frontend

Landing experience for NexusAds: paste a URL, confirm the brief, and export a ready-to-run ad pack.

## Scripts
- `npm start` for local development
- `npm run build` for production build
- `npm test` for tests

## Notes
- Frontend is currently a static product flow showcase.
- API origin is configurable via `REACT_APP_API_ORIGIN`.
