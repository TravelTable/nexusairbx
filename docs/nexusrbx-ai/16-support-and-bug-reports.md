## Before Reporting A Bug

Try these steps first:

1. Restart Roblox Studio.
2. Reopen NexusRBX Ai.
3. Check whether the plugin needs an update.
4. Copy the exact error message.
5. Test whether the problem happens in a new baseplate.

## Reporting Generated Code Issues

If generated code does not work, include:

- The original prompt.
- The generated code.
- The script type.
- The script location.
- The Output error.
- What you expected to happen.
- What actually happened.
