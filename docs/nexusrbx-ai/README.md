# NexusRBX Ai Documentation System

This folder contains paste-ready Markdown documentation for the NexusRBX Ai Roblox Studio plugin.

NexusRBX Ai is positioned as an AI-assisted Roblox Studio development tool. It helps Roblox creators generate Luau starting points, debug scripts, understand code, and prepare Studio-ready suggestions. It is currently in beta, and generated code should be reviewed, edited, and tested before publishing a Roblox experience.

## Getting Started

NexusRBX Ai helps Roblox developers move faster inside Roblox Studio without removing the need to understand, review, and test their work.

1. Install NexusRBX Ai from the Roblox Creator Store.
2. Open Roblox Studio and find NexusRBX Ai in the Plugins tab.
3. Open the plugin and complete the sign-in or account connection flow: `[Insert exact account connection steps]`.
4. Connect the plugin to the NexusRBX web app if required: `[Insert pairing flow or remove if not required]`.
5. Describe what you want the script to do, where it should go, and what script type it should use.
6. Review the generated Luau before inserting or copying it.
7. Test the result in Roblox Studio Play mode and watch the Output window for errors.
8. Ask follow-up prompts to improve, explain, or debug the result.
