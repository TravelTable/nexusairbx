# First-Time Setup

This page explains what to do the first time you open NexusRBX Ai in Roblox Studio.

## Open NexusRBX Ai

1. Open Roblox Studio.
2. Open a place.
3. Go to the Plugins tab.
4. Click NexusRBX Ai.
5. Wait for the plugin panel or window to load.

If the plugin opens but stays blank or loading, see [Troubleshooting](13-troubleshooting.md).
